Lo que entendemos del proyecto

Migración completa de la estructura de un sitio WordPress existente hacia Shopify, replicando diseño, navegación y contenido con fidelidad total.

🔍 Análisis del proyecto

El cliente cuenta con un sitio web funcional en WordPress y necesita trasladar toda su estructura a Shopify como plataforma base para su e-commerce. El objetivo no es un rediseño, sino una migración fiel: el nuevo sitio en Shopify debe verse y comportarse igual que el actual, conservando la identidad visual, la arquitectura de navegación y todo el contenido existente (páginas estáticas, blog, etc.). La carga de productos del catálogo será realizada de forma independiente por el equipo interno del cliente una vez que la estructura base esté en producción, por lo que la carga de productos no forma parte de este alcance.

01

Reto: Fidelidad del diseño

Replicar el diseño visual del sitio WordPress —tipografías, paleta de colores, layout, espaciados y componentes— dentro del ecosistema de Shopify usando Tailwind CSS para una implementación precisa y mantenible.

02

Reto: Estructura de navegación

Mapear y reconstruir todos los menús, enlaces internos y jerarquía de páginas del sitio original. Shopify maneja la navegación de forma diferente a WordPress, por lo que se requiere planificación cuidadosa.

03

Reto: Migración de contenido

Extraer y migrar todas las páginas estáticas (About, Contacto, Políticas, etc.) y entradas de blog desde WordPress, adaptando el contenido al sistema de Shopify sin pérdida de información.

04

Reto: Funcionalidades nativas

Identificar widgets, formularios de contacto, secciones dinámicas u otros elementos del WordPress actual y encontrar el equivalente nativo o la aplicación de Shopify correspondiente.

¿Qué incluye el proyecto?

Entregables detallados de la migración completa WordPress → Shopify.

🎨 Diseño y tema Shopify personalizado

  • Selección y configuración del tema base de Shopify más adecuado para replicar el diseño actual
  • Personalización completa con Tailwind CSS: colores, tipografías, espaciados y componentes visuales del sitio original
  • Implementación de header, footer y elementos de navegación global idénticos al sitio WordPress
  • Diseño responsivo conservado: mobile, tablet y desktop

📄 Migración de páginas estáticas

  • Extracción de todo el contenido de páginas estáticas del WordPress actual
  • Recreación de cada página en Shopify con el mismo layout, estructura y contenido
  • Páginas típicas: Inicio, Nosotros, Contacto, Políticas de privacidad, Términos y condiciones, FAQ, y demás páginas presentes en el sitio

📝 Migración del Blog

  • Exportación de todas las entradas de blog desde WordPress
  • Importación y configuración en el blog nativo de Shopify
  • Conservación de categorías, etiquetas, imágenes y formato de cada entrada
  • Configuración del listado de blog y página de entrada individual con el mismo diseño

🛒 Estructura base del e-commerce

  • Configuración de colecciones y estructura de catálogo lista para recibir productos
  • Páginas de colección, página de producto y carrito de compras con diseño fiel al sitio original
  • Configuración de métodos de pago y envío (estructura base, sin configuración operacional)
  • Nota: La carga de productos no está incluida — será realizada por el equipo interno del cliente

⚙️ Configuración técnica

  • Conexión del dominio existente a la nueva tienda Shopify
  • Configuración de redirecciones 301 para preservar el SEO del sitio actual
  • Instalación y configuración de apps esenciales (formulario de contacto, etc.)
  • Google Analytics / Tag Manager integrado si el sitio actual lo tiene
