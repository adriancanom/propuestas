Ecosistema Carnet Inteligente

Plataforma integral de seguridad y logística escolar para el Instituto Bilingüe Eliel, combinando rastreo GPS de estudiantes con gestión inteligente de recogida.

🔍 Lo que entendemos del proyecto

El Instituto Bilingüe Eliel necesita un ecosistema tecnológico que brinde tranquilidad a los padres y optimice la logística de salida del plantel. El sistema se basa en dispositivos GPS M16 personalizados (carnets inteligentes) que los estudiantes portan, permitiendo rastreo en tiempo real. La solución se complementa con una aplicación móvil para padres con módulo "Voy en Camino" (Smart Pickup) que calcula ETA basado en tráfico real, un panel de control web para el colegio conectado a plataformas profesionales de rastreo (Wialon/Navixy), y un dashboard público de salida para TV que muestra el estatus de recogida en tiempo real. El contexto geográfico es Panamá, con integración de mapas y capas de tráfico local.

01

Rastreo GPS en tiempo real

Dispositivos M16 con protocolo TCP/IP, conectividad 4G LTE-M y frecuencia de reporte configurable (60s en movimiento / 10 min en reposo). Geocercas circulares automáticas en puntos críticos.

02

App móvil para padres

Aplicación iOS y Android con mapa en tiempo real, historial de rutas de 7 días, módulo Smart Pickup con ETA basado en tráfico, gestión de alertas y notificaciones push.

03

Panel de control del colegio

Web app conectada a la API de Wialon o Navixy con monitor de puerta, alertas de proximidad (<500m), gestión de salida digital y dashboard global de dispositivos.

04

Dashboard público de salida

Pantalla para TV con mapa de zona, lista dinámica de llamado (nombre, grado, estatus en camino/llegando/en puerta) y validación visual con foto del estudiante.

05

Smart Pickup con ETA

Sistema que transmite la ubicación GPS del padre al servidor, calcula tiempo estimado de arribo con tráfico real y notifica al colegio automáticamente cuando está en proximidad.

06

Alertas y monitoreo avanzado

Alerta de desvío de ruta (>150m del eje), notificaciones de batería baja (<20%), sistema predictivo de carga (alerta dominical si <40%), y botón SOS reconvertido a "check-in de tranquilidad".

Marketing VIP® Componentes
Componente A — Panel de Control del Colegio

API y dashboard web conectado a Wialon o Navixy para la gestión completa del rastreo GPS y logística de salida del plantel.

🖥️ Web App — Panel del Colegio

  • Integración API Wialon/Navixy: Conexión a la plataforma de rastreo profesional para recibir datos de posición, estado de batería, señal y alertas de los dispositivos M16
  • Monitor de Puerta: Lista dinámica de alumnos cuyos padres están en proximidad, con alertas visuales y auditivas al entrar en radio de 500 metros
  • Gestión de Salida Digital: Registro de entrega de cada estudiante con timestamp, nombre del responsable y validación
  • Dashboard de Dispositivos: Vista global de todos los IMEIs activos, estado de conexión, nivel de batería y calidad de señal 4G
  • Geocercas Circulares: Configuración de zonas "Colegio" y "Casa" con alertas automáticas de entrada/salida
  • Alerta de Desvío de Ruta: Detección cuando un dispositivo se desvía más de 150m del eje del transporte escolar durante horario de salida
  • Perfiles de Administrador: Dashboard global para gestión de IMEI, soporte técnico y estados de conexión

📺 Pantalla Pública de Salida (Dashboard TV)

La pantalla pública de salida se implementará como una adaptación de la web app, diseñada para mostrarse a pantalla completa en un computador conectado a TV en el área de salida del plantel.

  • Mapa de zona: Vista con iconos de proximidad de padres en tiempo real
  • Lista de llamado dinámica: Nombre y grado · Estatus (En camino / Llegando / En Puerta) · Foto del estudiante
  • Auto-refresh: Actualización automática sin intervención manual
  • Modo kiosko: Optimizado para visualización en pantalla grande sin controles visibles

Componente B — Aplicación para Padres

App nativa para iOS y Android con rastreo en tiempo real, módulo Smart Pickup y gestión completa de alertas.

📱 App iOS & Android — Padres de Familia

  • Login seguro: Vinculación obligatoria de cuenta de usuario con IMEI único del dispositivo M16
  • Rastreo en tiempo real: Mapa con actualización cada 30-60 segundos de la ubicación del estudiante
  • Historial de rutas: Registro de trayectos de los últimos 7 días con reproductor visual
  • Módulo "Voy en Camino" (Smart Pickup): Activación manual de recogida, transmisión de GPS del celular al servidor, cálculo automático de ETA con tráfico real
  • Geocercas: Alertas automáticas de entrada/salida en zonas "Colegio" y "Casa"
  • Estado del hardware: Visualización de nivel de batería (0-100%) y calidad de señal 4G del dispositivo
  • Notificaciones Push: Alertas inmediatas para todos los eventos críticos del sistema
  • Gestión de alertas: Menú de ajustes con toggles independientes para geocercas, batería y desviaciones
  • Botón "Check-in de Tranquilidad": Notificación proactiva al padre: "✅ [Nombre] reporta que todo está bien" (ideal para excursiones)
  • Sistema Predictivo de Carga: Notificación los domingos a las 7:00 PM si el carnet tiene menos del 40% de batería

Marketing VIP® Notas
Notas importantes

Aclaraciones sobre el alcance del proyecto, dependencias externas y funcionalidades excluidas.

📡 Hardware: Dispositivo M16

La homologación del dispositivo M16 con la plataforma de rastreo (Wialon o Navixy) depende directamente del proveedor de la plataforma. Creemos que es factible, pero no depende de nosotros y no podemos garantizarlo. Se recomienda confirmar la compatibilidad con el fabricante o proveedor antes de iniciar el desarrollo. El costo del hardware y su configuración inicial no están incluidos en esta cotización.

📶 Reporte de Permanencia por WiFi (Indoor Positioning)

La funcionalidad de posicionamiento en interiores mediante WiFi Sniffing no puede ser cubierta con el hardware M16. Esta característica queda excluida del alcance de esta propuesta. En caso de requerirse en el futuro, se evaluaría con hardware adicional compatible.

📺 Pantalla Pública de Salida

El dashboard de salida para TV se implementará como una adaptación de la web app del panel de control, optimizada para visualización en pantalla completa desde un computador conectado a la televisión. No requiere desarrollo de software adicional independiente; se incluye dentro del componente A (Panel de Control del Colegio).

⚠️ Dependencias externas (no incluidas)

  • Dispositivos GPS M16 y su configuración de fábrica
  • Licencia/suscripción de la plataforma Wialon o Navixy
  • Cuenta de Google Maps API (con capa de tráfico de Panamá)
  • Publicación en Apple App Store y Google Play Store (cuentas de desarrollador)
  • Servidor/hosting para el backend (se puede cotizar aparte)
